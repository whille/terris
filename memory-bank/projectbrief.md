# 项目简介 - 俄罗斯方块网页版

## 项目概述
开发一个经典的俄罗斯方块（Tetris）网页版小游戏，使用现代Web技术栈实现。

## 核心要求
- **技术栈**: HTML、CSS、JavaScript（原生，无框架依赖）
- **游戏类型**: 经典俄罗斯方块
- **平台**: Web浏览器
- **响应式**: 支持桌面和移动设备

## 项目目标
1. 创建一个完全功能的俄罗斯方块游戏
2. 实现经典的游戏机制和规则
3. 提供良好的用户体验和视觉效果
4. 代码结构清晰，易于维护和扩展

## 成功标准
- 游戏可以正常运行，包含所有基础功能
- 代码质量高，结构清晰
- 用户界面美观，操作流畅
- 支持多种设备访问

## 约束条件
- 使用原生JavaScript，不依赖外部框架
- 代码需要在现代浏览器中运行
- 项目应该是自包含的，易于部署
